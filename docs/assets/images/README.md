# Images

This folder stores images used by the Jekyll site.

Suggested layout:

- `profile/`
  - personal photos such as your homepage or about-page image
- `projects/tep/`
  - diagrams, confusion matrices, plots, and other visuals for the Tennessee Eastman Process project

Example paths:

- `/assets/images/profile/faiaz.jpg`
- `/assets/images/projects/tep/mlp-confusion-matrix.png`
- `/assets/images/projects/tep/cnn-confusion-matrix.png`
- `/assets/images/projects/tep/process-diagram.png`

In markdown pages, reference them like this:

```md
![Faiaz](/assets/images/profile/faiaz.jpg)
```
